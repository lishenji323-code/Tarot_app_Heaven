# 神谕之境移动端原型

这是一款占卜治愈的手机端静态 Web App 原型，可直接打开 `index.html` 体验。

## 已实现

- 每日塔罗：每日固定一张牌，本地保存当天抽取状态。
- 雷诺曼解牌：支持 3 张和 5 张牌阵、洗牌动画、不重复抽牌、本地模拟治愈系解读。
- 评价组件：支持 0.5 星步进评分。
- 动态冥想：根据本地时间切换晨曦、日间、黄昏、夜间背景，包含 5 分钟倒计时、呼吸节奏和音量控件。
- 移动端 UI：底部导航、卡牌翻转、摊牌动画、异域纹样和卡通塔罗女神元素。

## 后续接入点

- 将 `app.js` 中的本地牌面数据替换为后端 API。
- 将本地模拟解牌函数 `makeAiReading` 替换为真实 AI 解牌接口。
- 将当前示意牌面替换为已授权的塔罗与雷诺曼牌面资源。
- 若要发布到应用商店，可迁移为 React Native 或 Flutter 项目。
